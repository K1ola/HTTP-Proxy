# HTTP-proxy

### Выполнено

- Проксирование HTTP запросов 5 баллов
- Отправка сохраненного запроса 5 баллов

#

### Инструкция

Склонировать репозиторий

```
git clone https://github.com/K1ola/HTTP-Proxy.git
cd HTTP-Proxy 
```

Запустить прокси
```
./bin/proxy
```
Запустить репитер
```
./bin/repeater 
```
